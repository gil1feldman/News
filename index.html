🔌 התחברות ל-iCount

הפרטים נשמרים אך ורק בדפדפן שלך (localStorage) תחת מפתח ייחודי למערכת זו.

✓ פרטים שמורים — ניתן לעדכן
איך להשיג API Token:
  1. התחבר ל-iCount → הגדרות → API
  2. לחץ "צור מפתח API חדש"
  3. העתק את ה-Token שמוצג (מופיע פעם אחת בלבד)
  4. את ה-CID תמצא ב: מערכת ← הגדרות ← החברה שלי — זה טקסט באותיות קטנות, לא מספר
